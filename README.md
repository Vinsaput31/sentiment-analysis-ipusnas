# User Sentiment Analysis of Ipusnas Application

## Overview
This project analyzes 9,791 user reviews from Google Play Store to understand user sentiment and identify key issues in the Ipusnas application.

## Objectives
- Analyze user sentiment distribution
- Identify app versions with the highest negative sentiment
- Compare sentiment and user ratings
- Provide actionable insights for app improvement

## Tools
- Python (Pandas, Scikit-learn, NLTK)
- Tableau

## Dataset
Source: Google Play Store  
Total Reviews: 9,791

## Key Insights
- 45% of reviews show negative sentiment
- Version 1.4.8 received the highest number of negative reviews
- Main issues: login problems, app errors, download failures
